Q1. What is Python? What are the ways to install Python in your PC?
Ans:
      Python is an interpreted, object-oriented, high-level programming language.
       It was developed by Guido Van Rossum in 1991.
       It is a popular language because it is easy to learn, it has less code to write compared to other languages, it’s 
       syntax is easier to read,etc.


       Ways to install python:
       1. Download it from www.python.org
       2. Download Anaconda Navigator which contains python and many other useful libraries including Jupyter
      


Q3. What is a Code Environment? And its types?
Ans:
       A code environment is a standalone and self-contained environment to run Python or R code.
       Code environment is a platform where we can write the code, compile it and can finally see the output of the   
       code in one place.


       Types of Code Environments:
       1. Text Editor - (ex-VS Code, Atom)
       2. Full IDE’s - (ex- PyCharm, Spyder)
       3. Notebook Environments - (ex- Jupyter Notebook)
